CONTROL PRESTAMOS APP - PROYECTO PARA ANDROID STUDIO

1. Descomprime este ZIP.
2. Abre Android Studio.
3. Selecciona "Open" y abre la carpeta ControlPrestamosApp_AndroidStudio.
4. Espera a que Gradle sincronice.
5. Si Android Studio te pide la ruta del SDK, configúrala.
6. Corre la app con un emulador o un teléfono Android.

Notas:
- Este paquete incluye el Excel de referencia: control_prestamos_2026A.xlsx
- El archivo local.properties fue omitido a propósito para que Android Studio lo regenere en tu PC.
- Si la app instalada da conflictos con la base local, desinstálala del dispositivo y vuelve a correrla.
